Пресейл‑конфигуратор

Быстрый расчёт по объекту, бюджетному коридору и типовым рыночным решениям: оборудование, зависимости, сводная спецификация и Excel-выгрузка.

Проектный расчёт в стилистике VIZHU

Приложение собирает конфигурацию из модулей, подсказывает рыночные ориентиры по брендам и сериям, считает состав оборудования, базовую себестоимость, предложение для клиента и экспортирует полный просчёт в таблицу.

Средний сегмент Рыночный профиль определяется по бюджету и позиционированию решения.

Проект

Название проекта попадает в шапку расчёта, правую сводку и имя выгружаемого файла.

Площадка и бюджет

Эти параметры влияют на рыночный сегмент, состав оборудования, монтажные коэффициенты и общий коридор стоимости.

Разделы проекта

Включай только те системы, которые действительно нужны в концепции. Для активных разделов появятся расчётные поля и рыночные ориентиры.

Конфигурация модулей

Внутри каждого раздела можно быстро уточнить ключевые параметры. Подбор брендов и ценовых ориентиров обновляется автоматически.

Сводная спецификация

Таблица формируется по текущей конфигурации и полностью уходит в Excel вместе с карточкой проекта, итогами и рисками.
