VIZHU — Presale Configurator v3

Новый конфигуратор объединяет нормализацию зон, внутреннюю инженерную спецификацию, клиентскую упаковку, коэффициенты маржи и шаблоны на основе образца Минтруда.

Инженерная модель → коммерческая модель → клиентский пресейл

Конфигуратор сначала собирает структуру объекта по зонам и механикам, затем считает внутреннюю спецификацию, а после этого упаковывает проект в клиентскую модель с распределённой маржой, как в реальном пресейл-файле. Встроен демо-шаблон по Минтруду, но логика остаётся универсальной для других объектов и входящих данных.

Паспорт проекта

Общие параметры влияют на global tier, коэффициенты и клиентскую подачу.

Площадка и коэффициенты

Эти параметры работают как глобальные множители для инженерки, монтажа и пресейл-упаковки.

Зоны и механики

Главная сущность нового конфига — сначала зоны и механики, потом оборудование. Для каждой зоны выбирается шаблон и интенсивность.

Проектные разделы

Enabled отвечает только за участие раздела в расчёте. Expanded отвечает только за раскрытие карточки.

Коммерческая модель и маржа

Маржа распределяется по категориям: hardware, content, integration, engineering, site-works. Можно менять профиль прямо в интерфейсе.

Спецификация

Есть два представления: внутренняя подробная спецификация и клиентская агрегированная версия, вдохновлённая приложенным пресейл-файлом.
